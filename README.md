Парсер рейтингов пользователей Teslacraft
-------
Программа автоматически собирает рейтинги пользователей форума Teslacraft и формирует топ-листы по положительным,
нейтральным и отрицательным рейтингам.

_Информация о рейтингах будет собрана, даже если пользователь закрыл профиль или был заблокирован_

Как работает
-----

1. Программа проходит по ID пользователей от 1 до указанного вами числа.
2. Собирает данные о рейтингах.
3. Сохраняет результаты в файл users.txt.
4. Формирует три файла с топами для каждого вида рейтингов:
    - users_positive_sort.txt
    - users_neutral_sort.txt
    - users_negative_sort.txt
5. Все ошибки записываются в last_log.log.

Как пользоваться
-----

1. Поместите .exe файл в отдельную папку _(рекомендуется)_.
2. Запустите .exe файл.
3. Введите количество пользователей, которых нужно обработать.
4. Укажите, сколько страниц можно открывать одновременно (для обеспечения максимальной скорости работы рекомендуется
   около 20).
5. Дождитесь завершения работы — файлы с результатами появятся в той же папке, где находится .exe файл.
